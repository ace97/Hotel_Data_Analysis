# Hotel Revenue Analysis

An ETL and Analysis project to find answers to stakeholder requirements on ways/areas to improve the business.
The project aims to help answer the questions.
1. Is the Hotel Revenue growing year by year?
2. What trends can we see in the data?
3. DO we need to increase our parking lot size?


Steps involved in building the solution:
1. Build a Database in MS SQL
2. Analyze and Retrieve Data with SQL
3. Connect Power BI to the Database and Load
4. Visualize Data in Power BI
